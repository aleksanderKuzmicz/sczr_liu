To run the application:
	1. create venv
		python3 -m venv <path_to_new_virtual_environment>

	2. activate venv:
		<path_to_new_virtual_environment>\Scripts\activate.bat

	2. install requirements
		pip3 install -r <path/to/project>\requirements.txt

	3. run one_app_to_rule_them_all.py 
		python3 <path_to_project>\src\one_app_to_rule_them_all.py
